CLASS POEM

Maria Sol Espona

Professor Li, Zhifang


Poem for my DART class

I'm in a new class called DART

For this one, I think you have to be very smart

Although I don't understand some parts,

I am going to put all my heart


I am trying to pay attention

because learning is my intention

This is my first time coding, did I mention?

And it feels like I'm entering the fourth dimension


I never thought I would learn how to code

and my mind is ready to blow

My slides now for some reason don't download

oh my, learning it's going to be a long road


Using the computer is something I keep avoiding

But I realize it only takes exploring

while my brain keeps loading

Man, I realize I love coding!


Thank you for enjoying my poem!
